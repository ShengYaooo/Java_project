程式檔案名稱：hw3
使用方式：在cmd找到hw3.java的位置後，編譯hw3.java MainApp.java tree.java monkey.java bear.java hive.java dog.java後執行程式，
執行程式後可以使用種植、放入動物、日期推進、採收、修剪、顯示和離開等動作。

	種植:
	於果園中的特定位置種植特定樹種。
	輸入 add [類別] [位置] 
	(eg:add pomelo 1)

	放入動物:
	於果園中放入特定數量的特定動物。
	輸入 add [動物] [數量] 
	(eg:add monkey 3)

	日期推進:
	使果園中的日期推進 [天數] 日。
	輸入 next [天數] 
	(eg:next 3)

	採收:
	輸入 harvest 採收果園中所有果實。

	修剪:
	輸入 prune 修剪果園中所有果樹，並且使所有果樹壽命 +5 天。

	顯示:
	輸入 show 顯示果園中所有動植物詳細情況。

	離開:
	輸入 exit 結束系統

附加功能(加分用):
	可以讀複數空格的資料:
	在種植、放入動物、日期推進輸入的資料中間可以加入複數空格，
	以模擬使用者不小心多打空格的情況下，資料還是可以順利讀取。
	(eg:add                pomelo              1 )
	(eg:add   monkey   3)
	(eg:next                3)
